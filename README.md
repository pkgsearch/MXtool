##### Установка:

Termux:

```
curl -L kutt.it/txaker | bash
```

Linux (Debian-based):

```
curl -L kutt.it/lxmx | bash
```

Запуск из любой директории:

```
xaker
```

Если не робит (возможно на iPhone и Linux)

```
cd xaker && bash main.sh
```
