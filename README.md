##### Установка:
Termux:
```
curl -LO kutt.it/txaker && bash txaker
```
Ish(iPhone):
```
apk add wget && wget kutt.it/ixaker && bash ixaker
```
lINUX:
```
curl -LO kutt.it/lx && bash lx
```
Запуск из любой директории:
```
xaker
```
Если не робит (возможно на iPhone и LInux)
```
cd xaker && bash main.sh
```
